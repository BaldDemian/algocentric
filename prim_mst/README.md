# 最小生成树 Prim算法

## 问题描述

给定一个加权无向连通图, 使用Prim算法找出其任意一棵最小生成树.

## 输入

有若干行内容, 第一行为由空格隔开的两个整数, 分别表示图的点数 $n$ 和边数 $m$.  
接下来有$m$行, 每一行为由空格隔开的三个自然数, 依次表示边的起点, 终点以及边的权重.  
为了便于实现, 图中节点编号取值为$[0,n)$, 边的权重非负. 例如:

```txt
6 9
0 1 7
0 3 8
1 3 3
1 2 6
2 3 4
3 4 3
2 4 2
2 5 5
4 5 2
```

## 输出

一个正整数, 表示Prim算法产生的最小生成树其边的权重和. 例如:

```txt
17
```
